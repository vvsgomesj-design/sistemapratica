🏁 Salvando Vidas no Trânsito!!!!!
Sistema de Evolução Teórico-Prática para Alunos e Instrutores de Condução.

Este projeto foi desenvolvido para auxiliar o processo de aprendizagem de novos condutores, oferecendo uma plataforma interativa com 9 módulos de vídeo (39 subfases) e um sistema de avaliação técnica baseado nas normas de trânsito.

🚀 Funcionalidades
Módulos de Vídeo: 39 lições práticas organizadas por fases, desde a preparação do veículo até à finalização da prova.

Controle de Progresso: O sistema salva automaticamente onde o aluno parou .

Área do Instrutor: Uma página dedicada para avaliar o desempenho do aluno em tempo real.

Geração de PDF: Emissão de um "Guia de Estudos" personalizado ou relatório de avaliação técnica.

Interface Responsiva: Visual moderno com "Dark Mode" para melhor leitura.

🛠️ Tecnologias Utilizadas
HTML5: Estrutura do site e incorporação de vídeo.

CSS3: Design moderno com variáveis de cores e Grid Layout.

JavaScript (Vanilla): Lógica de progressão, armazenamento local e interação com o utilizador.

html2pdf.js: Biblioteca para geração dos relatórios em PDF.
